How It Works

SM-2 Spaced Repetition

WordVault uses the same algorithm as Anki. Each time you review a card, you rate it Hard / OK / Easy. The app calculates when to show it next — the better you know a word, the longer it waits. This forces active recall, which is proven to be far more effective than re-reading.

4 Levels

New → just added. Learning → seen but still shaky. Review → scheduled for periodic review. Mastered → long interval, you know it well.
